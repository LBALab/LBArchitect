Common Nitro-Mecha-Penguin script

This script must be used for Nitro-Mecha-Penguin Actors. If is not, the Penguin will not work for the Scene.

The script includes part (comportement 1) which makes the Penguin pop out bonuses when his life goes below defined thresholds. That part is optional and can be removed.

The script has been copied from LBA 1.

Life Script:
comportement main
init_pingouin SELF
set_comportement 1
end_comportement
comportement 1
oneif life_point <= 45
give_bonus 0
endif
oneif life_point <= 30
give_bonus 0
endif
oneif life_point <= 15
give_bonus 0
endif
end_comportement
end
Editor-friendly version:
comportement main
  init_pingouin SELF
  set_comportement 1
end_comportement
comportement 1
  oneif life_point <= 45
    give_bonus 0
  endif
  oneif life_point <= 30
    give_bonus 0
  endif
  oneif life_point <= 15
    give_bonus 0
  endif
end_comportement
end
