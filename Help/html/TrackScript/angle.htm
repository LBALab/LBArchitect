Track Script: ANGLE (command)

ANGLE changes angle of facing for the actor.

Syntax:
  • ANGLE value(-32768 to 32767 integer)
Parameters:
  • value is the new absolute angle of facing. Value = 0 corresponds to 0°, value = 512 is 180° and value = 1024 is 360° (which is the same as 0°). Higher values will behave like modulo 1024 (1024 => 0, 1025 => 1, and so on).

1024 angle units are equal to 360°, so:

  • 1 angle unit is equal to about 0.35°,
  • 1 degree is about 2.85 angle units.

ANGLE will make the actor "visually" rotate from the old angle to the new one.

Rotation speed is determined by the Rotation delay Actor property.
