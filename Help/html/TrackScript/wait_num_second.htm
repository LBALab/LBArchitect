Track Script: WAIT_NUM_SECOND (command)

WAIT_NUM_SECOND pauses execution of the Track Script for specified number of seconds.

Syntax:
  • WAIT_NUM_SECOND value(0 to 255 integer)
Parameters:
  • value is the number of seconds to wait. The next Track Script line will be executed only after the time expires.
Example:
WAIT_NUM_SECOND 10 //wait 10 seconds
