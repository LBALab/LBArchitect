Introduction to LBA 1 Track Script

Track Script is the simpler of two kinds of Scripts that define behaviour of Scene Actors in LBA 1. Its main purpose is to provide a set of functions for use in Life Script, that make the Actor move in specified ways and produce specified sounds.

Initially the Track Script is stopped (however this depends on Actor control mode). It can be started only by calling SET_TRACK(_OBJ) Life Script command. Once started, the commands are executed one after another to the end or to a STOP command. Track Scripts of each Actor are run simultaneously.

Track Script commands by topic

Control statements:
  • END - marks end of the Track Script,
  • GOTO - jumps execution of the Script to specified Label,
  • LABEL - marks beginning of a block, provides an entry point to start the Script fragment from,
  • STOP - stops execution of current Track Script, used as end of a function,
  • WAIT_ANIM - pauses execution of the Script until the current Actor Animation ends,
  • WAIT_DOOR - pauses execution of the Script until the door Actor is fully closed or opened,
  • WAIT_NUM_ANIM - pauses execution of the Script until the Actor Animation has been played for specified number of times,
  • WAIT_NUM_SECOND - pauses execution of the Script for specified number of seconds,
Move and angle commands:
  • ANGLE - rotates the Actor to face specified angle,
  • ANGLE_RND - rotates the Actor by a semi-random angle,
  • BETA - immediately sets new angle for the Actor,
  • CLOSE - closes the door Actor,
  • FACE_HERO - rotates the Actor to face the Hero,
  • GOTO_POINT - makes the Actor head towards the specified Point,
  • GOTO_POINT_3D - makes the Sprite Actor move to specified Point,
  • GOTO_SYM_POINT - makes the Actor head back to the specified Point,
  • OPEN_DOWN, OPEN_LEFT, OPEN_RIGHT, OPEN_UP - open the door Actor,
  • POS_POINT - immediately moves the Actor to specified Point,
  • SPEED - sets movement speed for the Sprite Actor,
Appearance commands:
  • ANIM - selects a new Animation for the Actor,
  • BACKGROUND - enables/disables redrawing of the Actor,
  • BODY - selects a new Body for the Actor,
  • NO_BODY - removes the Actor's Body,
Sound and video commands:
  • PLAY_FLA - plays speified FLA movie,
  • REPEAT_SAMPLE - sets number of repetitions for the subsequent SIMPLE_SAMPLE command,
  • SAMPLE - plays specified sound sample,
  • SAMPLE_ALWAYS - plays specified sound sample looping it infinitely,
  • SAMPLE_RND - plays specified sound sample with randomly lowered pitch,
  • SAMPLE_STOP - stops playing specified sound sample,
  • SIMPLE_SAMPLE - plays sound sample specified number of times,
