Track Script: SIMPLE_SAMPLE (command)

SIMPLE_SAMPLE plays specified sound sample specified number of times.

Syntax:
  • SIMPLE_SAMPLE id(0 to 32767 integer)
Parameters:
  • id is index (in samples.hqr file) of the sound sample to play.

This command plays the specified sound sample number of times that has been previously set with the REPEAT_SAMPLE command.
