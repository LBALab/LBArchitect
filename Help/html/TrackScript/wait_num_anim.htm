Track Script: WAIT_NUM_ANIM (command)

WAIT_NUM_ANIM pauses execution of the Track Script until the current 3D Actor animation plays for spcified number of times.

Syntax:
  • WAIT_NUM_ANIM value(0 to 255 integer)
Parameters:
  • value is the number of animation repetitions. The next Track Script line will be executed only after the animation has been played for specified number of times.

This command will wait for the current animation to play specified number of times since the command has been executed. If WAIT_NUM_ANIM is called while an animation has been played for a number of times it will still wait the defined number of animations (not the defined number minus number of animations already played).

Example:
LABEL 1
ANIM 2 //do something
WAIT_NUM_ANIM 5 //wait for animation 2 to play 5 times
ANIM 1 //walk
GOTO_POINT 4
ANIM 0 //stop
STOP
END
