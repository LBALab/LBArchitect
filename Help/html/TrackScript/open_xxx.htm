Track Script: OPEN_DOWN (command)
Track Script: OPEN_LEFT (command)
Track Script: OPEN_RIGHT (command)
Track Script: OPEN_UP (command)

These commands open the door Actor in desired direction.

Syntax:
  • OPEN_DOWN distance(-32768 to 32767 integer)
  • OPEN_LEFT distance(-32768 to 32767 integer)
  • OPEN_RIGHT distance(-32768 to 32767 integer)
  • OPEN_UP distance(-32768 to 32767 integer)
Parameters:
  • distance is the distance the Actor should move by along the desired direction in Scene units.

These commands will open the door actor in desired direction. The actor must be a Sprite Actor and must use clipping. In the example below there is another command used for doors: WAIT_DOOR, which pauses execution of the script until the door is fully opened. WAIT_DOOR does not have to be used with opening commands, but without it the Script would continue execution even if the door was still moving.

Example:
LABEL 1
SAMPLE 35 //door moving sound
OPEN_LEFT 1550
WAIT_DOOR //wait for the door to close before executing next commands
STOP
END
