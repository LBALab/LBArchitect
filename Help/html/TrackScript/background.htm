Track Script: BACKGROUND (command)

BACKGROUND selects if the Actor is a background Actor.

Syntax:
  • BACKGROUND switch(0 or 1)
Parameters:
  • if switch = 1 the Actor will become a background Actor, if switch = 0 it will become a regular Actor.

If an Actor is a background Actor it is not redrawn all the time, but only once, when the screen is drawn. It is useful to save processor power, when the Actor's appearance doesn't change (it doesn't have an animation selected), it doesn't need to be refreshed too often. This command probably only works for Sprite Actors, and it works the same as Backgrounded Static Flag.

Example:
BACKGROUND 1 //set the Actor to be a background Actor
