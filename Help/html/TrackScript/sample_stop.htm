Track Script: SAMPLE_STOP (command)

SAMPLE_STOP stops playing the specified sound sample.

Syntax:
  • SAMPLE_STOP id(0 to 32767 integer)
Parameters:
  • id is index (in samples.hqr file) of the sound sample to stop.

This command stops playing the specified sound sample immediately. It also breaks the sound loop started with SAMPLE_ALWAYS command immediately (without waiting for the end of the sample).

It is perfectly fine to use SAMPLE_STOP for a sample that is not being played at the moment.
