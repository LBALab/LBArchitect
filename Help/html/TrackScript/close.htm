Track Script: CLOSE (command)

CLOSE closes the door Actor.

Syntax:
  • CLOSE

This command will close the door actor previously opened by OPEN_DOWN, OPEN_LEFT, OPEN_RIGHT or OPEN_UP command. The actor must be a Sprite Actor and must use clipping. In the example below there is another command used for doors: WAIT_DOOR, which pauses execution of the script until the door is closed. WAIT_DOOR doesn't have to be used with CLOSE, but without it the script will continue execution even if the door has not yet closed.

Example:
LABEL 1
SAMPLE 35 //door moving sound
CLOSE
WAIT_DOOR //wait for the door to close before executing next commands
STOP
END
