Track Script: BODY (command)

BODY selects new 3D body for the actor.

Syntax:
  • BODY value(-1 to 254 integer)
Parameters
  • value is virtual index of the new body for the actor. Body with specified virtual index must exist in the Actor's Entity. value = -1 selects no body for the Actor.
Example:
BODY 1 //select body 1 for the Actor
