Track Script: ANIM (command)

ANIM selects new animation for the 3D actor.

Syntax:
  • ANIM index(0 to 255 integer)
Parameters:
  • index is virtual index of the new animation for the Actor. Actor must be a 3D Actor (Is Sprite static flag must not be set). Animation with specified virtual index must exist in the Actor's Entity.
Example:
ANIM 1 //select animation 1 for the Actor
