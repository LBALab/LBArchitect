Track Script: SPEED (command)

SPEED changes movement speed of the Sprite Actor.

Syntax:
  • SPEED value(0 to 32767 integer)
Parameters:
  • value is the new speed. The larger value the higher speed the Sprite Actor will have. This command sets speed for GOTO_POINT_3D command. It does not influence OPEN_XXX or CLOSE commands.
Example:
SPEED 900 //set Actor's speed
GOTO_POINT_3D 1 //make the Actor move to the point
