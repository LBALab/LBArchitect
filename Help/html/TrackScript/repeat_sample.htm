Track Script: REPEAT_SAMPLE (command)

REPEAT_SAMPLE sets the number of repetitions for the subsequent SIMPLE_SAMPLE command.

Syntax:
  • REPEAT_SAMPLE count(0 to 32767 integer)
Parameters:
  • count is the number of repetitions.

This command does not play any sample, only sets number of repetitions for the following SIMPLE_SAMPLE command.
