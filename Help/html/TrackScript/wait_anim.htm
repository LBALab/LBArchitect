Track Script: WAIT_ANIM (command)

WAIT_ANIM pauses execution of the Track Script until the current 3D Actor animation finishes.

Syntax:
  • WAIT_ANIM

If WAIT_ANIM is not used, the next Track Script line will be executed immediately after setting the Actor's animation with the ANIM command. With the use of this command it is possible to wait with the next line until the animation is finished. This prevents changing the animation while previous animation is still executing, which would cause an unpleasant visual effect. It also makes possible to play an animation exactly once and then change it to another animation (this is shown in the example below).

Example:
LABEL 1
ANIM 2 //do something
WAIT_ANIM //wait for animation 2 to finish
ANIM 1 //walk
GOTO_POINT 4
ANIM 0 //stop
STOP
END
