Track Script: GOTO (control statement)

GOTO moves execution of the script to specified LABEL.

Syntax:
  • GOTO id(-1 to 254 integer)
  • GOTO label_name(text) (LBArchtect only)
Parameters:
  • id is the index of the target LABEL. Id = -1 makes the command work exactly like STOP (stops execution of the Script).
  • label_name is the name of the target LABEL.

Commands after GOTO cannot be executed unless they start with another LABEL statement.

Example:
LABEL 1
ANIM 1
GOTO walking_loop //Go to LABEL 103
LABEL 2 another_animation
ANIM 2
LABEL 103 walking_loop
GOTO_POINT 8
GOTO_POINT 9
GOTO_POINT 10
GOTO 103 //return to LABEL 103 (make a loop)
LABEL 105 stop_walking
STOP
END
