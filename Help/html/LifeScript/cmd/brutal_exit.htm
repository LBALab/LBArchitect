Life Script: BRUTAL_EXIT (command)

BRUTAL_EXIT ends the current game.

Syntax:
  • BRUTAL_EXIT

BRUTAL_EXIT immediately ends the current game and shows the main game menu. It behaves like if the player pressed Esc key and then chose Quit game.
