Life Script: SET_USED_INVENTORY (command)

SET_USED_INVENTORY enables auto-use of specified inventory item.

Syntax:
  • SET_USED_INVENTORY id(0 to 255 integer)
Parameters:
  • id is index of the inventory cell.

SET_USED_INVENTORY enabled specified inventory item to be used automatically under specific conditions. Normally the player can use an item from the inventory by pressing enter while having the item selected in the inventory. This action can be tested with USE_INVENTORY variable (it returns 1). If the auto-use is enabled for an item, the variable will always return 1 for that item without any actions from the player. This is commonly used for access cards - for example, if the player uses a card while being in specified Zone (near the card scanner), the SET_USED_INVENTORY will be called and when the player comes to that spot again, the card will be activated automatically (only if it is still in the inventory).

Once enabled the auto-use cannot be disabled, however it will not work when the card is not in the inventory (corresponding Game Flag unset).

Example:
if use_inventory 9 == 1
set_used_inventory 9 //enable auto-use for next times
// perform other actions
endif
