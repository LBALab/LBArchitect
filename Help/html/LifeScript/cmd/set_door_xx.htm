Life Script: SET_DOOR_DOWN (command)
Life Script: SET_DOOR_LEFT (command)
Life Script: SET_DOOR_RIGHT (command)
Life Script: SET_DOOR_UP (command)

These commands set the door Actor at specified position along desired direction.

Syntax:
  • SET_DOOR_DOWN distance(-32768 to 32767 integer)
  • SET_DOOR_LEFT distance(-32768 to 32767 integer)
  • SET_DOOR_RIGHT distance(-32768 to 32767 integer)
  • SET_DOOR_UP distance(-32768 to 32767 integer)
Parameters:
  • distance is the distance the Actor should be put by from the closed position along the desired direction in Scene units.

These commands will set the door position at the specified distance from the closed position. The actor must be a Sprite Actor and must use clipping. After loading the Scene all doors are in closed positions by default (the initial positions as seen in Architect or other Scene editor). These commands are used mainly to change the initial positions of the doors (to ensure the doors are open when the Hero walks out of a building). See common door script for an example usage.
