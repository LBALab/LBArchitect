Life Script: ADD_CHOICE (command)

ADD_CHOICE adds an option that player can choose during the next ASK_CHOICE call.

Syntax:
  • ADD_CHOICE index(0 to 32767 integer)
Parameters:
  • index is the mapped index of text line to be displayed for the choice. The line will also be read by the associated voice if player chooses it.

ADD_CHOICE command in connection with ASK_CHOICE command and CHOICE allows asking player questions, like "Do you want to buy that?" in a shop. All the possible answers must be first added with ADD_CHOICE command, then ASK_CHOICE must be executed, and after that the answer chosen may be checked with the CHOICE variable. The list of answers is cleared after the choice has been made, so it is not possible to ask two questions with the same list of answers only adding the answers once. In such case the answers must be added again for the second question.

The text indexes used for this command are not the text positions in an lbt file (even subfile of text.hqr). Correct indexes are mapped in the corresponding odd subfile of the text.hqr. LBArchitect takes this into consideration when displaying list of possible indexes for this command.

Example:
add_choice 1 //add text line 1 as first possible answer
add_choice 4 //second possible answer
add_choice 5 //third possible answer
ask_choice 7 //ask the question saying line 7
if choice == 5 //test if player have chosen line 5
cam_follow 1
endif
