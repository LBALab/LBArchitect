Life Script: PROJ_3D (command)
Life Script: PROJ_ISO (command)

PROJ_3D sets 3-D projection for the Scene, while PROJ_ISO sets isometric projection.

Syntax:
  • PROJ_3D
  • PROJ_ISO

Isometric projection should be always used for regular gamepaly. 3-D projection is only used for the credits sequence. 3-D projection switches the game to first person perspective view, but the game engine cannot render the terrain in such mode, so the terrain is all black.
