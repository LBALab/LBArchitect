Life Script: FOUND_OBJECT (command)

FOUND_OBJECT shows the new object animation and displays the message for it.

Syntax:
  • FOUND_OBJECT id(0 to 255 integer)
Parameters:
  • id is index of the object that is 'found'

This command just shows the message and object image rotating in the center of the screen. It does not add the item to the Hero's inventory. To add the item to inventory it's associated Game Flag must be also set.

Example:
found_object 3 //show that object 3 has been found
set_flag_game 3 1 //add object 3 to the inventory
