Life Script: SET_HOLO_POS (command)

SET_HOLO_POS marks specified position (arrow) on the Holomap.

Syntax:
  • SET_HOLO_POS id(0 to 255 integer)
Parameters:
  • id is index of the Holomap position to be shown.

Position marked with SET_HOLO_POS can be cleared at leater time by using CLR_HOLO_POS command. Holomap positions are defined in the 9th entry (starting with 1) of the ress.hqr file.
