Life Script: TEXT (command)

TEXT displays specified message in the top of the screen.

Syntax:
  • TEXT text_id(0 to 32767 integer)
Parameters:
  • text_id is the mapped index of text line to be displayed.

The text will be displayed at the top of the screen without any frames. This command is usually used for credits list sequence. The text will be cleared the next time that part of the screen is redrawn, or when CLEAR_TEXT command is used.

The text indexes used for this command are not the text positions in an lbt file (even subfile of text.hqr). Correct indexes are mapped in the corresponding odd subfile of the text.hqr. LBArchitect takes this into consideration when displaying list of possible indexes for this command.
