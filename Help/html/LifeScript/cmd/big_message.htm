Life Script: BIG_MESSAGE (command)

BIG_MESSAGE displays specified text message in full screen.

Syntax:
  • BIG_MESSAGE index(0 to 32767 integer)
Parameters:
  • index is the mapped index of text to be displayed. The text will also be read by the associated voice.

BIG_MESSAGE command displays text message in the big full screen frame, and plays sound sample assiciated with the text (voice that reads the text).

When the Actor is saying the message, balloon will be shown over its head. This can be changed with the BALLOON_ON/OFF commands.

The text indexes used for this command are not the text positions in an lbt file (even subfile of text.hqr). Correct indexes are mapped in the corresponding odd subfile of the text.hqr. LBArchitect takes this into consideration when displaying list of possible indexes for this command.

Example:
big_message 1 //display text 1
