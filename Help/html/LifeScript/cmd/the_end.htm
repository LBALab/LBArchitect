Life Script: THE_END (command)

THE_END shows credits list sequence.

Syntax:
  • THE_END

THE_END runs the ending credits list sequence (if should be invoked after the successful ending sequence). The sequence is made of Grid and Scene id 119 (counting from 0), if they don't exist the game will crash. After the sequence the game automatically shows the main menu.
