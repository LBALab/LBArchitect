Life Script: RESTORE_LAST_TRACK (command)

RESTORE_LAST_TRACK resumes execution of the Actor's Track Script.

Syntax:
  • RESTORE_LAST_TRACK

RESTORE_LAST_TRACK resumes execution of the Actor's Track Script from the point it has been previously stopped with the STOP_CURRENT_TRACK command.
