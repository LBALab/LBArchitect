Life Script: ANIM (command)
Life Script: ANIM_OBJ (command)
Life Script: ANIM_SET (command)

ANIM selects new animation for the current Actor, ANIM_OBJ selects new animation for the Actor specified in the first parameter, ANIM_SET selects new annimation immediately (without a smooth transition from the old animation).

Syntax:
  • ANIM index(0 to 255 integer)
  • ANIM_OBJ actor(0 to 255 integer) index(0 to 255 integer)
  • ANIM_SET index(0 to 255 integer)
Parameters:
  • index is a virtual index of the new animation for the Actor. Actor must be a 3D Actor (unset Is Sprite flag). Animation with specified virtual index must exist in the Actor's File 3D Entity.
  • actor is the index of the target Actor (the Actor must exist in the Scene).
Example:
anim 1 //select animation 1 for the current Actor
anim_obj 3 1 //select animation 1 for Actor 3
