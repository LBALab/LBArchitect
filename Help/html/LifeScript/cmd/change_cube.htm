Life Script: CHANGE_CUBE (command)

CHANGE_CUBE loads another Grid and Scene.

Syntax:
  • CHANGE_CUBE scene(0 to 255 integer)
Parameters:
  • scene is index of the target Grid (in lba_gri.hqr) file and it's associated Scene (in scene.hqr file).

CHANGE_CUBE causes the current Scene to fade out (to black), load Scene with specified index, and fade in. Unlike changing Scene with the Cube Zones, by using CHANGE_CUBE the Hero Actor will be in the default position for the target Scene.
