Life Script: BETA (command)

BETA immediately (without an Animation) changes angle of facing for the Actor.

Syntax:
  • BETA value(0 to 32767 integer)
Parameters:
  • value is the new angle of facing. Value = 0 corresponds to 0°, value = 512 is 180° and value = 1024 is 360° (which is the same as 0°). Higher values will behave like modulo 1024 (1024 => 0, 1025 => 1, and so on).

Unlike Track Script's ANGLE command, this command will immediately set the new angle for the actor - without an animation - regardless of the Rotation delay property.

1024 angle units are equal to 360°, so:

  • 1 angle unit is equal to about 0.35°,
  • 1 degree is about 2.85 angle units.
