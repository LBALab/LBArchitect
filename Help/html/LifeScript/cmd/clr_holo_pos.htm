Life Script: CLR_HOLO_POS (command)

CLR_HOLO_POS clears (unmarks) specified position (arrow) on the Holomap.

Syntax:
  • CLR_HOLO_POS id(0 to 255 integer)
Parameters:
  • id is index of the Holomap position to be cleared.

CLR_HOLO_POS clears the position that was prewiously marked with SET_HOLO_POS command. Holomap positions are defined in the 9th entry (starting with 1) of the ress.hqr file.
