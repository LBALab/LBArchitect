Life Script: GRM_OFF (command)

GRM_OFF hides disappearing Grid Fragments previously shown with SET_GRM command.

Syntax:
  • GRM_OFF

SET_GRM clears all Fragments that have been shown with the SET_GRM and Grid Zones. However, Grid Zone framgmens will be re-shown with the next screen refresh, so only a short flicker may be visible. See SET_GRM for more details on Fragments.
