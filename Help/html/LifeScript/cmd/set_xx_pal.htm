Life Script: SET_DARK_PAL (command)
Life Script: SET_NORMAL_PAL (command)

SET_XX_PAL commands change current game palette woithout fading.

Syntax:
  • SET_DARK_PAL
  • SET_NORMAL_PAL

These commands immediately change the current game palette to the dark or normal palette. Changed palette does not influence the FADE_XX commands, for example if current plaette is dark palette the FADE_PAL_ALARM will still fade from normal palette to alarm, not from dark to alarm.

Regular (normal) game palette

Dark palette - used for building interiors during night
