Life Script: OBJ_COL (command)

OBJ_COL selects if the Actor can collide with other Actors.

Syntax:
  • OBJ_COL switch(0 or 1)
Parameters:
  • if switch = 1 the Actor will collide with other Actors, if switch = 0 it will be able to pass through other Actors.

OBJ_COL changes value of the Can collide with Objects static flag. When it is set, other Actors (Objects) are solid to the Actor. When it is not set, the Actor can pass through other Actors. Both Actors must have this flag set to be able to collide with each other.
