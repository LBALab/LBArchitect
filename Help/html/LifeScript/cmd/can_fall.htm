Life Script: CAN_FALL (command)

CAN_FALL selects if the Actor is influenced by gravity.

Syntax:
  • CAN_FALL switch(0 or 1)
Parameters:
  • if switch = 1 the Actor will fall when he has got no ground under feet, if switch = 0 it will walk in the air.

CAN_FALL changes value of the Can fall static flag. If it is set, the Actor falls when there are no solid Bricks under its feet. If the fall disatance is too high it may lose some life points or even die. This setting requires the Can collide with Bricks flag to be also set.
