Life Script: SET_TRACK (command)
Life Script: SET_TRACK_OBJ (command)

SET_TRACK runs the Actor's Track Script from specified LABEL, SET_TRACK_OBJ does the same for another Actor.

Syntax:
  • SET_TRACK label_id(-1 to 254 integer)
  • SET_TRACK label_name(text)
  • SET_TRACK_OBJ actor(0 to 255 integer) label_id(-1 to 254 integer)
  • SET_TRACK_OBJ actor(0 to 255 integer) label_name(text)
Parameters:
  • label_id is the index of the Track Script LABEL to start execution from; label_id = -1 will stop the Track Script execution,
  • label_name is the name of the Track Script LABEL to start execution from (label names are optional),
  • actor is the index of the target Actor (the Actor must exist in the Scene).

By using -1 as the label_id the Track Script can be stopped. This works similar to STOP_CURRENT_TRACK, except that in this case RESTORE_LAST_TRACK cannot be used to run Track Script again from where it stopped.

Examples:
set_track go_there //reference label by name
set_track 101 //reference label by index
