Life Script: PLAY_CD_TRACK (command)

PLAY_CD_TRACK plays specified CD music track.

Syntax:
  • PLAY_CD_TRACK track(0 to 255 integer)
Parameters:
  • track is the CD track index to be played

PLAY_CD_TRACK uses different track indexes than the Music ID Scene property. This command uses the indexes of the Tracks as they appear on the CD. Track 0 is the data track, thus it cannot be played, and can be used to stop the music (PLAY_CD_TRACK 0).
