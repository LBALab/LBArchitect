FADE_XX commands change current game palette using fading.

Syntax:
  • FADE_ALARM_PAL
  • FADE_ALARM_RED
  • FADE_PAL_ALARM
  • FADE_PAL_RED
  • FADE_RED_ALARM
  • FADE_RED_PAL

Each of these commands fade the screen from the palette speicified in the middle segment of the commmand name (ALARM, PAL or RED) to the palette specified in the last segment. For example FADE_ALARM_PAL fades from alarm palette to the normal game palette.

Regular game palette

Alarm palette - used for alarm indication

Red palette - used only to fade to other palettes

Each fading takes about one second, and during the transition all screen movement is paused (including Actor animations). If a new Scene is loaded the normal palette will be restored.

Example:
fade_pal_red //fade to red
fade_red_alarm //and immediately fade to alarm palette
