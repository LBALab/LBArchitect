Life Script: SET_DIRMODE (command)
Life Script: SET_DIRMODE_OBJ (command)

SET_DIRMODE sets the control (direction) mode for the Actor, SET_DIRMODE_OBJ sets the control mode for another Actor.

Syntax:
  • SET_DIRMODE mode [ f_actor(0 to 255 integer)]
  • SET_DIRMODE actor(0 to 255 integer) mode [ actor(0 to 255 integer)]
Parameters:
  • mode can be one of the following:
    • NO_MOVE
    • MANUAL
    • FOLLOW
    • TRACK
    • SAME_XZ
    • RANDOM
  • f_actor is index of the Actor to follow (the Actor must exist in the Scene). This parameter is required only for FOLLOW and SAME_XZ modes,
  • actor is index of the Actor to set the mode for (the Actor must exist in the Scene).

For description of control modes see Actors.

Examples:
set_dirmode no_move
set_dirmode follow 0
set_dirmode 1 follow 0
