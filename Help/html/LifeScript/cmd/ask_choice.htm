Life Script: ASK_CHOICE (command)
Life Script: ASK_CHOICE_OBJ (command)

ASK_CHOICE asks player the specified question with answers defined using preceding ADD_CHOICE commands, ASK_CHOICE_OBJ does the same, but displays balloon for the Actor specified in the first parameter.

Syntax:
  • ASK_CHOICE index(0 to 32767 integer)
  • ASK_CHOICE_OBJ actor(0 to 255 integer) index(0 to 32767 integer)
Parameters:
  • index is the mapped index of text line to be displayed for the question. The line will also be read by the associated voice.
  • actor is the index of the target Actor (the Actor must exist in the Scene).

ASK_CHOICE(_OBJ) command allows asking player questions, like "Do you want to buy that?" in a shop. To use it, all the possible answers must be first added with ADD_CHOICE command. The list of answers is cleared after the choice has been made, so it is not possible to ask two questions with the same list of answers only adding the answers once. In such case the answers must be added again for the second question. Further the line chosen by player can be tested with CHOICE variable.

When the Actor is asking the question, balloon will be shown over its head. This can be changed with the BALLOON_ON/OFF commands.

The text indexes used for this command are not the text positions in an lbt file (even subfile of text.hqr). Correct indexes are mapped in the corresponding odd subfile of the text.hqr. LBArchitect takes this into consideration when displaying list of possible indexes for this command.

Example:
add_choice 1 //add text line 1 as first possible answer
add_choice 4 //second possible answer
add_choice 5 //third possible answer
ask_choice 7 //ask the question saying line 7
if choice == 5 //test if player have chosen line 5
cam_follow 1
endif
