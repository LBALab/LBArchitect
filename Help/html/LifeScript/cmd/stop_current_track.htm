Life Script: STOP_CURRENT_TRACK (command)

STOP_CURRENT_TRACK stops execution of the Actor's Track Script.

Syntax:
  • STOP_CURRENT_TRACK

Track Script execution may be resumed from the last point using the RESTORE_LAST_TRACK command.
