Life Script: BRICK_COL (command)

BRICK_COL selects if the Actor can collide with Bricks.

Syntax:
  • BRICK_COL value(0 to 2)
Parameters:
  • if value = 0 the Actor will be able to pass through Bricks, otherwise it will collide with Bricks. value = 2 also enables Low collision static flag (otherwise it will be disabled).

BRICK_COL changes value of the Can collide with Bricks static flag. When it is set, the walls and Grid objects are solid to the Actor. When it is not set, the Actor can pass through walls and Grid objects, although it cannot fall or go up (the Can fall flag is ignored, the Actor stays at a constant vertical position).
