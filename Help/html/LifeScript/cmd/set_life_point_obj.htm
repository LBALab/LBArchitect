Life Script: SET_LIFE_POINT_OBJ (command)

SET_LIFE_POINT_OBJ sets number of life points for specified Actor to specified value.

Syntax:
  • SET_LIFE_POINT_OBJ actor(0 to 255 integer) points(0 to 255 integer)
Parameters:
  • actor is the index of the target Actor (the Actor must exist in the Scene)
  • points is the number of life points to set for specified Actor

SET_LIFE_POINT_OBJ sets life point amount for the Actor to the specified value. If the new value is 0, the Actor will die. There is no version of this command working on the current Actor (without '_OBJ' suffix), but the index of current Actor may be specified manually, also in LBArchitect self keyword can be used to reference the current Actor.
