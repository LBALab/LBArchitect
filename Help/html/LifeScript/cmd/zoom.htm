Life Script: ZOOM (command)

ZOOM turns sceneric zoom on or off.

Syntax:
  • ZOOM switch(0 or 1)
Parameters:
  • if switch = 1 the scene will be zommed-in (2x magnification), if switch = 0 normal view will be used.

ZOOM works only if it has been enabled in the game settings. In zoom mode the Hero (or other Actor set by CAM_FOLLOW command) is tracked by the camera in real-time.
