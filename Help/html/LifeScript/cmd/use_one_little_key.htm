Life Script: USE_ONE_LITTLE_KEY (command)

USE_ONE_LITTLE_KEY takes one little key from the Hero.

Syntax:
  • USE_ONE_LITTLE_KEY

Little keys are used to let the Hero access a specific room only if he has at least one little key. Little keys are lost when another Scene is loaded, so they cannot be moved between areas. Current number of little keys held by the Hero can be tested with NUM_LITTLE_KEYS variable.
