Life Script: ENDIF (control statement)

ENDIF marks the end of a conditional Script block.

Syntax:
  • ENDIF

Conditional statements allow for testing variables of the world against given values and executing specified Script pieces depending whether the condition is met or not. The end of the Script block must end with ENDIF statement, even if it consists of a single line only.

Example:
if zone == 1
add_fuel 4
else
cam_follow 0
endif

if zone == 2
full_point
endif
