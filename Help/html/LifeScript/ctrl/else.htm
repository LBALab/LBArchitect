Life Script: ELSE (control statement)

ELSE is placed before a Script block that will be executed if the corresponding conditional statement is not met.

Syntax:
  • ELSE

Conditional statements allow for testing variables of the world against given values and executing specified Script pieces depending whether the condition is met or not. The end of the Script block must end with ENDIF statement, even if it consists of a single line only. ELSE is an addition to IF, ONEIF and SWIF statements to provide an alternative Script block to the block under condition.

Example:
if zone == 1 //if current Actor is inside Zone 1
add_fuel 4
else //if current Actor is NOT inside Zone 1
cam_follow 0
endif
