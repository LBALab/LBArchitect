Life Script: END_LIFE (control statement)

END_LIFE terminates execution of the Actor's Life Script.

Syntax:
  • END_LIFE

END_LIFE stops execution of the Actor's Life Script. It does not influence the Actor itself - it is still present in the Scene, and can move. The Track Script of the Actor will be still executing. Futher the Life Script can be run again with SET_COMPORTEMENT_OBJ command called from another Actor.
