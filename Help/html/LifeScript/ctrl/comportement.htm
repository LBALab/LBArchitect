Life Script: COMPORTEMENT (control statement)

COMPORTEMENT marks the beginning of a comportement.

Syntax:
  • COMPORTEMENT name(text)
Parameters:
  • name is an unique identifier of the comportement. It may be used in SET_COMPORTEMENT(_OBJ) statement.

Comportements are loops (similar to states in a state machine) in Life Script that constantly execute code that is inside them. When the execution goes to the END_COMPORTEMENT statement it loops back to the first command of the comportement. Moving execution to another comportement is possible with SET_COMPORTEMENT(_OBJ) statement. See Life Script overview for more details.

Example:
comportement 1
cam_follow 1
set_comportement 2
end_comportement
comportement 2
midi_off
end_comportement
end
