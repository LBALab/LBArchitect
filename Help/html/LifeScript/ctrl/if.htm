Life Script: IF (control statement)

IF begins conditional statements, that allow executing particular piece of code only if some conditions are met.

Syntax:
  • IF variable [actor(0 to 255 integer)] operator value
Parameters:
  • variable is name of the world property to be evaluated. For list of variables see Life Script overview.
  • actor is index of the Actor to test the variable for. This parameter is required only for some variables (mostly the ones that end with _OBJ).
  • operator can be one of the following:
    ==equal
    !=not equal
    >greater than
    <less than
    >=greater or equal
    <=less or equal
  • value is the value to compare the variable with. Data type of the value depends on the variable used. The value must be constant, it cannot be a variable.

Conditional statements allow for testing variables of the world against given values and executing specified Script pieces depending whether the condition is met or not. The end of the Script block must end with ENDIF statement, even if it consists of a single line only. IF is the simplest statement, that allows for one condition test. In order to tests more conditions and allow execution if any one of them is met (boolean OR) the additional OR_IF statement must be used. To do the same only when all of the conditions are met (boolean AND) the IF statements must be nested (as shown in the example below).

Example:
if zone == 1 //test if current Actor is inside Zone 1...
if fuel >= 4 //... AND Hero's fuel amount is at least 4 units
cam_follow 0
endif
endif
