Life Script: END_COMPORTEMENT (control statement)

END_COMPORTEMENT marks the end of a comportement.

Syntax:
  • END_COMPORTEMENT

Comportements are loops (similar to states in a state machine) in Life Script that constantly execute code that is inside them. When the execution goes to END_COMPORTEMENT statement it loops back to the first command of the comportement. Moving execution to another comportement is possible with SET_COMPORTEMENT(_OBJ) statement. See Life Script overview for more details.

Example:
comportement 1
cam_follow 1
set_comportement 2
end_comportement
comportement 2
midi_off
end_comportement
end
