Life Script: RETURN (control statement)

RETURN makes the Life Script execution return to the beginning of the active comportement.

Syntax:
  • RETURN

RETURN works exactly like END_COMPORTEMENT, but it can be used inside the comportement (commands after it still belong to the same comportement, and it can be used inside a conditional block).

It is commonly used to prevent execution of the remaining part of the comportement when the comportement is changed (see SET_COMPORTEMENT).
