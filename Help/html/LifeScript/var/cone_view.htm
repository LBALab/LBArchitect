Life Script: CONE_VIEW (variable)

CONE_VIEW returns distance to specified Actor if it is in specific view angle.

Syntax:
  • if CONE_VIEW actor(0 to 255 integer) == distance(0 to 32000 integer)

(if may be also: or_if, swif, oneif; == may be also: !=, >, <, >=, <=)

Parameters:
  • actor is index of the target Actor (the Actor must exist in the Scene).
Return value:
  • CONE_VIEW returns distance to specified Actor if it is in -45°+45° range to the current Actor's face. If target Actor is 0 (the Hero), the angle range is ignored, and the variable will always return distance between the Actors.

CONE_VIEW will return 32000 if the target Actor is outside the -45°+45° angle range.

Examples:
if cone_view 2 < 3000 //if current Actor 'sees' Actor 2 in -45°+45° field and is less thn 3000 units away from it
//perform operations
endif
