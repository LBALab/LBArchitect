Life Script: DISTANCE (variable)
Life Script: DISTANCE_3D (variable)

DISTANCE returns distance between the current Actor and another Actor in X and Z axes, DISTANCE_3D also takes Y (upright) axis into consideration.

Syntax:
  • if DISTANCE actor(0 to 255 integer) == value(0 to 32000 integer)
  • if DISTANCE_3D actor(0 to 255 integer) == value(0 to 32000 integer)

(if can be also: or_if, swif, oneif; == can be also: !=, >, <, >=, <=)

Parameters:
  • actor is the index of the target Actor (the Actor must exist in the Scene).
Return value:
  • DISTANCE and DISTANCE_3D return distance in Scene units between the current Actor and Actor specified in the parameter.

It is not recommended to use equality operator (==), as the condition will most likely never be met.

Example:
if distance 3 <= 1500 //if current Actor is 1500 units or less away from the Actor 3
//perform operations
endif
