Life Script: NUM_LITTLE_KEYS (variable)

NUM_LITTLE_KEYS tells how many little keys the Hero has.

Syntax:
  • if NUM_LITTLE_KEYS == value(0 to 255 integer)

(if may be also: or_if, swif, oneif; == may be also: !=, >, <, >=, <=)

Return value:
  • NUM_LITTLE_KEYS returns number of little keys the Hero has.

Little keys are used to open locked doors (this behaviour must be programmed in Scripts). The number of little keys is reset to zero while changing Scenes.

Example:
if num_little_keys >= 1 //test if the Hero has at least 1 little key
//perform operations
endif
