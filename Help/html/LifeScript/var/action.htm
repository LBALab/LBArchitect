Life Script: ACTION (variable)

ACTION returns 1 if the player is using the Action command.

Syntax:
  • if ACTION == value(0 or 1)

(if may be also: or_if, swif, oneif; == may be also: !=, >, <, >=, <=)

Return value:
  • ACTION returns 1 if the player is using the Action command at the moment, 0 otherwise.

Usually there is no need to worry that player will press the Action key for a short while, and the Script will be doing something else at the moment and will not notice that. The scripts are looped thousands of times per second, unless a Script is very long and complex (which is not recommended in general).

Example:
if action == 1 //if player is 'doing Action'
//perform operations
endif
