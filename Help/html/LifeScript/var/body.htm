Life Script: BODY (variable)
Life Script: BODY_OBJ (variable)

BODY returns body index for the current Actor, BODY_OBJ returns body index for the Actor specified in the parameter.

Syntax:
  • if BODY == index(-1 to 254 integer)
  • if BODY_OBJ actor(0 to 255 integer) == index(-1 to 254 integer)

(if may be also: or_if, swif, oneif; == may be also: !=, >, <, >=, <=)

Parameters:
  • actor is the index of the target Actor (the Actor must exist in the Scene).
Return value:
  • BODY returns virtual index of the body that is currently assigned to the current Actor, or -1 if the Actor currently does not have any Body,
  • BODY_OBJ returns virtual index of the body that is currently assigned to the specified Actor, or -1.

Operators > and >= should be used with care, because the game engine treats Body index = -1 as 255 (unsigned byte), so a conditional statement like BODY > 10 will be true if the Actor's Body is -1.

Examples:
if body <= 3 //if current body is less or equal to 3
//perform operations
endif

if body_obj 5 == 1 //if current body of Actor 5 is equal to 1
//perform operations
endif
