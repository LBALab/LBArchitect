Life Script: HIT_BY (variable)

HIT_BY returns id of an Actor that is hitting the current Actor.

Syntax:
  • if HIT_BY == index(0 to 255 integer)

(if may be also: or_if, swif, oneif; == may be also: !=, >, <, >=, <=)

Return value:
  • HIT_BY returns index of the Actor that is hitting the current Actor.

If the Actor is not getting hit by any other Actor, HIT_BY will return -1, but this case cannot be tested. Although the Actor index value is one byte, so 255 should be equal to -1, the engine keeps the value as a 4-byte (dword) integer internally, and -1 as dword (232) is not equal to -1 as byte (255). Thus if the Actor is not getting hit by any Actor, the condition will be false for any comparison value.

This variable will return id of the hitting Actor exactly once per every hit.

Example:
if hit_by == 1 //if current Actor is getting hit by Actor 1
//perform operations
endif
