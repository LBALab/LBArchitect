Life Script: BEHAVIOUR (variable)

BEHAVIOUR returns current Hero behaviour.

Syntax:
  • if BEHAVIOUR == value(0 to 3 integer)

(if may be also: or_if, swif, oneif; == may be also: !=, >, <, >=, <=)

Return value:
  • BEHAVIOUR returns index of the current Hero behaviour. The possible values are:
    • 0 = Normal
    • 1 = Athletic
    • 2 = Aggressive
    • 3 = Discrete
Example:
if behaviour == 1 //if the Hero is in Athletic behaviour
//perform operations
endif
