Life Script: USE_INVENTORY (variable)

USE_INVENTORY returns 1 if specified inventory item has been used by the player. It also displays the item animation if so.

Syntax:
  • if USE_INVENTORY item(0 to 255 integer) == active(0 or 1)

(if may be also: or_if, swif, oneif; == may be also: !=, >, <, >=, <=)

Return value:
  • USE_INVENTORY returns 1 if the specified item has ben used, 0 otherwise.

USE_INVENTORY returns 1 if the player has used the inventory item, that is if he has pressed Enter key while having selected the item in the inventory, or if the auto-usage of the item has been enabled (see SET_USED_INVENTORY command). If the variable returns 1 it will also display small item animation in the top left screen corner.

USE_INVENTORY works only for items that are present in the inventory. It does not work for empty inventory spaces. Adding and removing an item from the invetory can be done by setting corresponding Game Flag (see SET_FLAG_GAME command). Game Flags from 0 to 27 are reserved for inventory items.

The Scripts are executing very fast, the current Comportement is looped hundreds of times per second (faster than the player would release the key), so it is certain that using an item it will be noticed by the Script.

It makes no sense to check the variable against other values than 1, or use other operators than == and !=.

Example:
if use_inventory 2 == 1 //test if player has used item 2
//perform operations
endif
