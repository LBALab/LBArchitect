Life Script: ANIM (variable)
Life Script: ANIM_OBJ (variable)

ANIM returns animation index for the current Actor, ANIM_OBJ returns animation index for the Actor specified in the parameter.

Syntax:
  • if ANIM == index(0 to 255 integer)
  • if ANIM_OBJ actor(0 to 255 integer) == index(0 to 255 integer)

(if may be also: or_if, swif, oneif; == may be also: !=, >, <, >=, <=)

Parameters:
  • actor is the index of the target Actor (the Actor must exist in the Scene).
Return value:
  • ANIM returns virtual index of the animation that is currently assigned to the current Actor,
  • ANIM_OBJ returns virtual index of the animation that is currently assigned to the specified Actor.
Examples:
if anim >= 1 //if current animation is greater or equal to 1
//perform operations
endif

if anim_obj 5 == 1 //if current animation of Actor 5 is equal to 1
//perform operations
endif
