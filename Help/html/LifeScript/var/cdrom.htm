Life Script: CDROM (variable)

CDROM returns 1 if the game CD is in the drive

Syntax:
  • if CDROM == value(0 or 1)

(if may be also: or_if, swif, oneif; == may be also: !=, >, <, >=, <=)

Return value:
  • CDROM returns 1 if the game CD is in the drive

CDROM is of little use nowadays, because LBAWin engine constantly checks for the CD, and doe not allow to continue playing if it is removed from the drive. It could be useful for the old engine (that didn't require CD to play) or modified engines.
