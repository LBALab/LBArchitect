Life Script: CHAPTER (variable)

CHAPTER indicates the story progress.

Syntax:
  • if CHAPTER == value(0 to 255 integer)

(if may be also: or_if, swif, oneif; == may be also: !=, >, <, >=, <=)

Return value:
  • CHAPTER returns current value of the internal chapter variable.

CHAPTER value can be increased using INC_CHAPTER command. Chapters are used to logically organize the game story, and easier modify Actor behaviour depending on the story progress.

Example:
if chapter >= 3 //if the chapter is 3 or higher
//perform operations
endif
