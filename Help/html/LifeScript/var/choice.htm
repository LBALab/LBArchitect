Life Script: CHOICE (variable)

CHOICE tells what answer the player has chosen after executing ASK_CHOICE(_OBJ) command.

Syntax:
  • if CHOICE == value(0 to 32767 integer)

(if may be also: or_if, swif, oneif; == may be also: !=, >, <, >=, <=)

Return value:
  • CHOICE returns id of the text that has been chosen.

CHOICE variable in connection with ASK_CHOICE and ADD_CHOICE commands allows asking player questions, like "Do you want to buy that?" in a shop. All the possible answers must be first added with ADD_CHOICE command, then ASK_CHOICE must be executed, and after that the answer chosen may be checked with the CHOICE variable.

The player cannot cancel the selection. If he presses Escape key, the last answer will be chosen.

The text indexes used for this variable are not the text positions in an lbt file (even subfile of text.hqr). Correct indexes are mapped in the corresponding odd subfile of the text.hqr. LBArchitect takes this into consideration when displaying list of possible indexes for this variable.

Example:
add_choice 1 //add text line 1 as first possible answer
add_choice 4 //second possible answer
add_choice 5 //third possible answer
ask_choice 7 //ask the question saying line 7
if choice == 5 //test if player have chosen line 5
//perform operations
endif
