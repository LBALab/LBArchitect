Life Script: CARRIED_BY (variable)

CARRIED_BY returns id of the Actor that carries current Actor.

Syntax:
  • if CARRIED_BY == value(0 to 255 integer)

(if may be also: or_if, swif, oneif; == may be also: !=, >, <, >=, <=)

Return value:
  • CARRIED_BY returns index of the Actor that carries the current Actor.

If the Actor does not stand on any other Actor, CARRIED_BY will return -1, but this case cannot be tested. Although the Actor index value is one byte, so 255 should be equal to -1, the engine keeps the value as a 4-byte (dword) integer internally, and -1 as dword (232) is not equal to -1 as byte (255). Thus if the Actor does not stand on any Actor, the condition will be false for any comparison value.

Example:
if carried_by == 3 //if the Actor is carried by Actor 3
//perform operations
endif
